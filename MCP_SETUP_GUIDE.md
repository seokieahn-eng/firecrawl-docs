# MCP Setup Guide

This guide provides instructions to set up the MCP environment.

## Prerequisites
- Node.js (version x.x.x)
- NPM (version x.x.x)
- Git

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/seokieahn-eng/firecrawl-docs.git
   ```
2. Navigate to the project directory:
   ```bash
   cd firecrawl-docs
   ```
3. Install the necessary packages:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Troubleshooting
If you encounter issues, consider the following:
- Ensure Node.js and NPM are installed.
- Check your internet connection.
- Review error messages for guidance.

## Conclusion
Following these instructions should help you set up the MCP environment successfully. If issues persist, please consult the official documentation or seek assistance from the community.
